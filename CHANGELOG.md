## OW 1.0.9
*`2023/5/1`*

***更改日志***

    · 重构拼音补全引擎，现在支持模糊多音字。

    · 重构语法高亮引擎，提供精准着色。

***开发者注释***

    现在每个字的多种读音都将纳入联想范围。
    例如 查莉娅 可以输入 ZhaLiYa, ChaLiYa, ZhaLY, CLY 等显示。

    启用 LSP DocumentSemanticTokens 作为主要着色引擎，TextMate 引擎已废弃。
    新引擎实时分析语法，不正确的语法将中断高亮，并在底部状态栏显示细节。

***高亮作用域***

    属性        代表        示例
    --------------------------------
    comment     注释        //...
    string      字符串      "{0} {1}"
    number      数字        0
    variable    变量        A
    symbol      符号        >=
    action      动作        创建效果
    condition   条件        按钮被按下
    constant    常量        安娜
    event       事件        持续 - 全局
    other       其它        禁用

***主题 JSON***

    "semanticHighlighting": true,
	"semanticTokenColors": {
		"comment": {
			"foreground": "#000000"
		},
		"string": {
			"foreground": "#000000"
		},
		"number": {
			"foreground": "#000000"
		},
		"variable": {
            "foreground": "#000000",
			"fontStyle": "italic"
		},
		"symbol": {
			"foreground": "#000000"
		},
		"action": {

## OW 1.0.3

*`2023/4/21`*

***更改日志***

    · 新增 快捷面板按钮，用于切换自动换行行为。

    · 修复 数组参数的补全问题。

    · 修复 整理无法处理设置英雄选项中堡垒"切换模式 冷却时间"的问题。

***开发者注释***

    插件的整理功能(格式化)会清洗非必需换行符。
    例如，一个多行嵌套数组会被整理为一行。

    使用自动换行功能可在不污染源码基础上将数组展开为多行。
